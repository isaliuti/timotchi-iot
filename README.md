# timotchi-iot
Este trabalho apresenta uma proposta de solução em IoT para monitorar o tempo que indivíduos permanecem em frente a telas. O objetivo é conscientizar sobre os riscos do uso excessivo e incentivar hábitos mais saudáveis. Quando o usuário permanece mais de uma hora contínua diante da tela, o sistema emite um alerta, estimulando uma pausa.
